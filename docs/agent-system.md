# Agent System

NarrativeSite AI uses agents as role-based coordinators and skills as reusable capabilities.

## MVP Agents

- `agents/orchestrator.md`: owns stage, state, routing, and user questions.
- `agents/interview-agent.md`: legacy detailed interview role.
- `agents/story-strategy-agent.md`: combines story intelligence, career impact, and life themes.
- `agents/media-design-agent.md`: combines media curation, design direction, and motion direction.
- `agents/site-planning-agent.md`: turns approved strategy into a site plan and copy brief.
- `agents/copywriting-agent.md`: legacy focused copy role.
- `agents/build-qa-agent.md`: builds, reviews, and patches.

The original twelve-agent files are retained as expanded specialist references. The MVP runtime should start with the smaller agent set and split only when a specialist role becomes necessary.
