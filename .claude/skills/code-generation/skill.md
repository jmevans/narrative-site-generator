# Code Generation Skill

## Purpose

Build or patch the deployable personal website from approved artifacts.

## Preferred Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Method

1. Start from `website-template`.
2. Store content in data/config files.
3. Build semantic, responsive components.
4. Implement reduced-motion support.
5. Optimize media usage and avoid exposing private data.
6. Run checks when a runtime exists.

## Output

```json
{
  "file_tree": [],
  "changed_files": [],
  "setup_steps": [],
  "run_commands": [],
  "deployment_notes": []
}
```
