# QA Review Skill

## Purpose

Review the site plan or generated website for credibility, accessibility, privacy, performance, usability, and conversion clarity.

## Checks

- Unsupported claims
- Overstated accomplishments
- Missing proof
- Generic positioning
- Accessibility issues
- Performance risks
- Motion overload
- Mobile usability problems
- Privacy issues
- CTA clarity

## Output

```json
{
  "overall_score": 0,
  "critical_issues": [],
  "recommended_fixes": [],
  "unsupported_claims": [],
  "accessibility_notes": [],
  "performance_notes": [],
  "privacy_notes": [],
  "ready_to_publish": false
}
```
