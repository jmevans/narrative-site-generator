# Design Direction Skill

## Purpose

Create an emotionally resonant, credible, accessible visual identity for the user's personal website.

## Method

1. Read audience, goal, story themes, media assets, and tone preferences.
2. Propose 3 distinct themes with rationale.
3. Define colors, typography, layout, components, CTA style, and accessibility notes.
4. Avoid generic premium styling that is not grounded in the user's actual story.

## Rules

- Do not rely on a single-hue palette.
- Avoid generic dark-gradient tech aesthetics unless justified.
- Keep contrast accessible.
- Use media and proof as primary visual signals.
