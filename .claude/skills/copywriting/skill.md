# Copywriting Skill

## Purpose

Write concise, accurate, public-facing personal website copy.

## Rules

- Use confirmed facts or clearly marked placeholders.
- Avoid buzzwords and inflated claims.
- Make headlines specific.
- Keep paragraphs short and skimmable.
- Preserve the user's credibility over dramatic language.
