# Motion Direction Skill

## Purpose

Define purposeful motion that supports story, hierarchy, and interaction.

## Method

1. Choose a motion style name and principles.
2. Define entrance, scroll, hover, metric, media, and transition animations.
3. Include reduced-motion alternatives for every animation.
4. Prefer CSS or Framer Motion for standard behavior. Use GSAP only for advanced storytelling needs.

## Rules

- Motion should reveal, guide, or clarify.
- Avoid constant motion and childish easing.
- Keep animation performant and accessible.

## Output

```json
{
  "motion_style_name": "",
  "principles": [],
  "animations": [],
  "media_motion_plan": []
}
```
