# Site Architecture Skill

## Purpose

Turn story, media, copy needs, and design direction into a clear site structure.

## Method

1. Define the primary visitor path.
2. Select homepage sections and their order.
3. Assign content blocks, media assets, animation behavior, and CTAs.
4. Decide which material belongs in expandable details or case-study pages.
5. Document mobile behavior and content gaps.

## Output

```json
{
  "site_map": [],
  "navigation": [],
  "interaction_model": "",
  "mobile_notes": [],
  "content_gaps": []
}
```
